Supervillain App

The Supervillain Directory is a full-stack web app built with Python (Flask) that lets users create, view, and manage a collection of fictional villains. Designed with full CRUD functionality, it supports adding, displaying, and deleting entries via forms and RESTful API routes. The frontend uses HTML, CSS, and Jinja2 to render dynamic villain data in styled cards. The backend uses Flask and SQLAlchemy to manage a SQLite database and define API endpoints using jsonify for JSON responses. The app features dynamic content updates, custom error messaging, and timestamping via datetime. Key highlights include RESTful routing, clean UI layout, and structured code across the stack. Quotivation App

This front-end quote curation app is built with React, allowing users to explore, filter, and save inspirational quotes with a smooth, interactive interface.
